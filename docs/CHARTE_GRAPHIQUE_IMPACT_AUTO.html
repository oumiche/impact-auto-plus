Utilisation des couleurs

  • Bleu Principal : Navigation, titres, éléments de navigation
  • Bleu Secondaire : Sous-titres, éléments secondaires
  • Or Accent : Boutons d'action, liens, éléments importants
  • Gris Clair : Arrière-plans, cartes, surfaces
  • Gris Bordure : Bordures, séparateurs, lignes
  • Texte Principal : Contenu, paragraphes, texte général
